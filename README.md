# R-fundamental-and-data-analyzation

## Practice 1 — Student Score Data Analysis

This R practice demonstrates basic data analysis using an Excel dataset of student scores. The code:

- Reads an Excel file using `readxl`
- Previews the first few rows of the dataset
- Checks the structure and column types
- Generates summary statistics (min, max, mean, etc.)
- Accesses specific columns and calculates averages
- Filters rows based on conditions (e.g., Age > 23)
- Selects specific columns after filtering

## Practice 2 — Filtering
Filtering rows & columns covers filtering by a single condition, multiple conditions with &, selecting columns by name or index, and combining both row and column filtering at once.

## Practice 3 — Loops
Loops cover a for loop over a sequence, a for loop over a vector, accumulating a sum with for, a basic while loop with a counter, and a practical while loop that halves a value repeatedly.

## Practice 4 — Functions
Functions cover a basic function with return(), default argument values, returning multiple values via a list, and a practical function that combines filtering and summarizing from Practice 1.
